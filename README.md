# PocketFlow

**Privacy-first personal finance tracking from SMS messages**

PocketFlow automatically extracts transaction data from SMS messages, classifies merchants, and provides intelligent budget tracking and financial insights—all on your device.

## 🚀 Quick Start

### Prerequisites
- Flutter SDK (3.0+)
- Android SDK or Xcode for iOS
- Dart 3.0+

### Installation

```bash
# Clone the repository
git clone https://github.com/ShankarSomu/PocketFlow.git
cd PocketFlow

# Install dependencies
flutter pub get

# Run the app
flutter run
```

## ✨ Key Features

- **SMS Intelligence**: Automatic transaction detection from SMS messages
- **Smart Classification**: ML-powered merchant categorization
- **Account Management**: Track multiple accounts and transfers
- **Budget Tracking**: Monitor spending across categories
- **Privacy-First**: All processing happens locally on your device
- **Export & Backup**: Export data in multiple formats (CSV, JSON, Excel, PDF)

## 📱 Usage

1. **Grant SMS Permission**: Allow the app to read financial SMS
2. **Auto-Detection**: Transactions are automatically extracted
3. **Review & Categorize**: Confirm or adjust transaction details
4. **Track & Analyze**: View budgets, trends, and insights
